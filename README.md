📝 Simple Notepad

A clean, lightweight, ad-free notepad built with HTML, CSS, and JavaScript.

Features

- 📝 Simple note editor
- 💾 Auto Save (Local Storage)
- 📋 Copy Notes
- 🗑️ Clear Notes
- 📱 Mobile Friendly
- 🌐 Works Offline
- 🚫 No Ads
- 🔒 No Login Required

Technologies Used

- HTML5
- CSS3
- JavaScript

Usage

1. Open the app in your browser.
2. Start typing.
3. Your notes are saved automatically.
4. Reopen the app anytime to continue where you left off.
